## 2026-04-09 02:26 视频管家 v4.0 AI-Native设计方案

### 竞品深度分析

#### 影库管家（VideoButler）
- 影视爱好者本地媒体中心
- 核心：AI智能分类、TMDB元数据刮削、语义搜索、局域网分享
- 设计思维：一站式媒体中心（采集→播放）

#### Eagle
- 设计师素材管理神器
- 核心：浏览器插件采集、智能文件夹、颜色筛选、动作自动化
- 设计思维：效率至上，每个操作都考虑快捷键和批量处理

#### Billfish
- 国产免费素材管理
- 核心：非侵入式管理（索引/复制/剪切三种模式）、Qt轻量、零迁移成本
- 设计思维：不打扰用户，尊重原有文件结构

#### 零泉
- 免费全能素材管理
- 核心：AI图片放大、Midjourney提示词管理、调色板导出、设计软件联动
- 设计思维：设计师工作流集成，不只是管理更是创作工具

### 用户确认的核心场景
- **场景**：A. 个人视频整理（家庭录像/下载的电影）
- **文件处理**：综合方案，索引+可执行（默认不碰原文件）
- **AI介入**：A. 我说你做（被动执行）
- **优先级**：自然语言交互、搜索速度、去重准确性、人脸识别

### v4.0 AI-Native架构

#### 三层架构
```
用户层：自然语言 + 可视化界面
    ↓
智能层：Kimi API（意图理解、任务规划、异常处理）
    ↓
执行层：Kimi Claw + 视频管家（文件操作、视频分析、数据库存储）
```

#### 核心功能模块
1. **自然语言命令系统** - 支持"扫描下载文件夹""找重复视频"等自然语言指令
2. **非侵入式文件管理** - 索引模式（默认）+ 标记模式（sidecar文件）+ 整理模式（需确认）
3. **四层去重系统** - L1文件大小+时长 → L2快速pHash → L3详细pHash序列 → L4文件MD5
4. **花名册人脸识别** - 专为家庭场景优化，支持"可能人物"提示、忽略路人、时间范围过滤
5. **多维度搜索** - 人物/时间/路径/重复/组合条件
6. **Web可视化界面** - 统计卡片、快速操作、花名册筛选、视频瀑布流

#### 差异化定位
> "最懂你的AI视频管家 - 像助理一样对话，像网盘一样安全"

| 优势对比 | 视频管家 | 竞品 |
|---------|---------|------|
| 人脸识别 | 花名册学习模式（更准） | 影库管家（弱） |
| 视频去重 | 多层pHash（独家） | Eagle（无） |
| AI理解 | Kimi自然语言（主动） | Billfish（纯索引） |
| 视频专精 | pHash+人脸识别 | 零泉（通用素材） |

### 实施路线图
- Phase 1（1周）：视频管家API化、SQLite设计、基础NL解析
- Phase 2（1周）：Kimi API接入、命令映射、反馈生成
- Phase 3（1周）：Web界面、缩略图缓存、搜索筛选
- Phase 4（3天）：测试调优、文档、打包
