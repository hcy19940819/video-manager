# 2026-04-08 - VideoFaceManager 开发记录

## 完成的工作

### VideoFaceManager 项目
创建了视频人物识别管理系统，参考影库管家(VideoButler)设计。

#### 已创建的文件结构
```
VideoFaceManager/
├── cli.py / cli_hybrid.py    - 命令行工具
├── search_engine.py          - 搜索引擎（以图搜视频、语义搜索、高级筛选）
├── manager.py                - Web管理面板
├── workflow_hybrid.py        - 一键工作流
├── README.md / README_HYBRID.md / README_VIDEOBUTLER.md - 文档
└── core/
    ├── database.py           - SQLite数据库
    ├── hybrid_library.py     - 混合人物库管理
    ├── hybrid_recognizer.py  - 人脸识别引擎
    └── video_learner.py      - 反向学习模块
```

#### 核心功能实现
1. **反向学习** - 从已分类视频中自动提取人物特征
2. **主动录入** - 从照片导入人物特征
3. **纠正反馈** - 人工指正提高识别准确率
4. **以图搜视频** - 上传照片搜索包含该人物的视频（影库管家核心功能）
5. **语义搜索** - 用自然语言描述搜索视频
6. **高级筛选** - 按时长、人物数量等多维度过滤

#### 技术栈
- 人脸识别：RetinaFace + ArcFace
- 硬件加速：ONNX Runtime with DirectML（支持AMD RX 6750XT）
- 数据库：SQLite
- Web框架：Flask

## 用户环境
- 显卡：AMD RX 6750XT（非NVIDIA，不支持CUDA）
- 系统：Windows
- 使用场景：管理小网红视频，需要按人物分类

## 关键决策
- 采用DirectML而非CUDA以支持AMD显卡
- 设计双线路综合模式：反向学习+主动录入+纠正反馈
- 参考影库管家设计核心功能（以图搜视频、语义搜索）
- 全中文界面和文档（用户不懂英文）
