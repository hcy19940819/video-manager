# 视频管家 - 一体化视频管理系统

**一次扫描，数据互通**

整合了视频去重 + 人物识别 + 以图搜视频功能。

---

## 特点

- ✅ **扫一次，全搞定** - 同时生成去重指纹和人物识别数据
- ✅ **数据互通** - 去重库和人物库通过视频路径关联
- ✅ **增量扫描** - 只处理新增视频，不重复扫旧的
- ✅ **全中文** - 命令和输出都是中文

---

## 安装

### 1. 安装依赖

```bash
cd videomanager
pip install opencv-python pillow numpy insightface tqdm
```

### 2. 初始化系统

```bash
python 视频管家.py 初始化
```

会自动创建：
- 数据/去重库.db
- 数据/人物库.db
- 数据/截图/

---

## 使用

### 第一步：扫描视频（核心）

```bash
# 第一次扫描
python 视频管家.py 扫描 "/视频文件夹"

# 以后新增视频，只扫新增的
python 视频管家.py 扫描 "/视频文件夹" --增量
```

扫描时会同时做：
1. 提取视频信息（时长、分辨率）
2. 计算指纹（用于去重）
3. 识别人脸（用于人物分类）
4. 保存到两个数据库

### 第二步：查找重复

```bash
python 视频管家.py 找重复
```

输出示例：
```
发现 8 组重复视频
预计可释放空间: 5.2 GB
报告已保存: 重复报告.md
```

### 第三步：查看人物库

```bash
python 视频管家.py 人物库
```

### 第四步：从已分类视频学习

如果你有已经按人物分好类的视频：

```
已分类视频/
├── 张三/
│   ├── 视频1.mp4
│   └── 视频2.mp4
└── 李四/
    └── 视频3.mp4
```

运行：
```bash
python 视频管家.py 学习 "/已分类视频"
```

系统会学习每个人物的特征，用于识别新视频。

### 第五步：以图搜视频

```bash
python 视频管家.py 搜图 "/照片/张三.jpg"
```

上传照片，搜索包含该人物的视频。

## 完整工作流

```bash
# 1. 初始化（只做一次）
python 视频管家.py 初始化

# 2. 扫描所有视频（扫一次，数据都有了）
python 视频管家.py 扫描 "/我的视频"

# 3. 找重复
python 视频管家.py 找重复

# 4. 如果有已分类的视频，让系统学习
python 视频管家.py 学习 "/已分类视频"

# 5. 以后新增视频，增量扫描
python 视频管家.py 扫描 "/我的视频" --增量

# 6. 查看统计
python 视频管家.py 统计
```

---

## 命令列表

| 命令 | 作用 |
|------|------|
| `初始化` | 创建数据库和目录 |
| `扫描 <路径>` | 扫描视频，生成去重+人物数据 |
| `扫描 <路径> --增量` | 只扫描新增的视频 |
| `找重复` | 查找重复视频 |
| `人物库` | 显示人物库内容 |
| `学习 <路径>` | 从已分类视频学习人物特征 |
| `搜图 <图片路径>` | 以图搜视频 |
| `统计` | 显示统计信息 |

## 数据库说明

### 去重库
存储视频指纹，用于找重复：
- 视频路径
- 文件大小
- 快速哈希（aHash）
- 文件哈希（MD5前1MB）
- 时长、分辨率

### 人物库
存储人物识别结果：
- 视频路径
- 人物名
- 出现时间点
- 置信度
- 人脸截图路径

**两个库通过"视频路径"关联。**

## 与旧版的关系

| 旧版工具 | 新版整合 |
|---------|---------|
| videodedup_fast.py | 视频管家.py 找重复 |
| VideoFaceManager | 视频管家.py 扫描 + 人物库 |

**新版优势**：
- 一次扫描，数据给两个功能用
- 不用扫两遍视频
- 数据库互通

---

## 注意事项

1. **第一次扫描较慢** - 需要解码视频、计算哈希、识别人脸
2. **增量扫描很快** - 只处理新增的文件
3. **人脸识别的准确性** - 依赖学习数据，已分类视频越多越准
4. **截图保存位置** - 数据/截图/人物名/
