# 📖 视频管家 v4.0 - 使用手册

## 🚀 快速开始

### 1. 安装与初始化

```bash
# 下载并安装
pip install video-manager

# 初始化系统
python 视频管家.py init

# 配置AI（输入你的Kimi API Key）
python 视频管家.py config --api-key your-api-key
```

### 2. 第一次使用

打开浏览器访问 `http://localhost:5000`，或直接命令行对话：

```bash
# 启动Web界面
python 视频管家.py web

# 或命令行对话
python 视频管家.py "帮我整理视频"
```

## 👥 花名册（人物管理）

### 什么是花名册？

花名册是视频管家的人物识别系统。通过学习已分类的视频，系统能自动识别新视频中的人物。

### 建立花名册步骤

**Step 1: 准备已分类视频**
```
已分类视频/
├── 小宝/              ← 这个文件夹里全是小宝的视频
│   ├── 生日会.mp4
│   └── 公园玩.mp4
├── 爷爷/              ← 这个文件夹里全是爷爷的视频
│   └── 下棋.mp4
└── 奶奶/
    └── 做饭.mp4
```

**Step 2: 让系统学习**
```bash
python 视频管家.py "学习已分类视频文件夹里的人物"
```

或Web界面：
1. 进入"👤 人物"页面
2. 点击"➕ 添加新人物"
3. 选择"从文件夹学习"
4. 选择 `/已分类视频` 文件夹

**Step 3: 查看学习结果**
- 系统自动提取每个人物的特征
- 显示"已学习 5 个人物"
- 每个人物显示代表头像

### 使用花名册识别新视频

```bash
python 视频管家.py "识别未分类文件夹里的人物"
```

系统会自动：
1. 扫描新视频
2. 检测人脸
3. 匹配花名册
4. 给视频打上人物标签

### 管理人物

| 操作 | 方法 |
|------|------|
| 查看某人的视频 | 点击花名册头像 |
| 修改人物名字 | 点击"编辑" |
| 删除人物 | 点击"删除"（不影响视频） |
| 补充学习 | 上传更多该人物的照片/视频 |

## 🔍 去重功能

### 检测原理（四层去重）

视频管家使用四层检测，从快到精：

1. **文件大小+时长** - 秒级排除明显不同的视频
2. **快速pHash** - 检测相似视频（64位指纹）
3. **详细pHash序列** - 检测剪辑版本（关键帧序列）
4. **MD5文件哈希** - 确认完全重复（前1MB）

### 执行去重

```bash
python 视频管家.py "找重复视频"
```

系统会显示：
- 重复组数
- 每组相似度（100% = 完全重复，<100% = 相似）
- 可节省空间

### 处理重复视频

对于每组重复，你有三个选择：

| 操作 | 说明 |
|------|------|
| ✅ 保留 | 标记为保留，其他为删除候选 |
| 🗑️ 删除 | 移动到回收站（30天可恢复） |
| 🚫 忽略 | 标记为不重复，不再提示 |
| 📂 打开文件夹 | 在资源管理器中查看 |

### 智能建议

系统会给出建议：
- **完全重复**：删除较小/较新的那个
- **相似视频**：可能是剪辑版，建议都保留
- **系列视频**：同一场景不同角度，建议保留

## 🛡️ 数据安全

### 不碰原文件原则

视频管家默认**不移动、不删除、不重命名**你的原视频文件。所有操作只在数据库中记录元数据。

实际文件保持不变：
```
/Videos/家庭录像/生日会.mp4    ← 原文件（不动）
/Videos/家庭录像/公园玩.mp4    ← 原文件（不动）
```

所有数据存储在：
```
~/.video-manager/
├── video_master.db      # 数据库
├── thumbnails/          # 缩略图缓存
├── faces/               # 人脸特征
└── config.json          # 配置文件
```

### 删除操作保护

- 删除视频 → 移动到系统回收站（30天可恢复）
- 清理数据库 → 不影响原文件
- 卸载软件 → 视频仍在原处

### 数据备份

**自动备份**
- 每周自动备份数据库到 `~/.video-manager/backup/`

**手动备份**
```bash
python 视频管家.py "备份数据"
# 生成 backup_20240815.zip
```

**恢复数据**
```bash
python 视频管家.py "恢复数据 backup_20240815.zip"
```

## ⚙️ 高级设置

### 调整识别精度

```bash
# 人脸检测阈值
python 视频管家.py config --face-threshold 0.7

# 选项：0.5(宽松) 0.7(标准) 0.9(严格)
```

### 去重阈值设置

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| 快速pHash阈值 | 汉明距离 | 5-10（越小越严格）|
| 详细pHash阈值 | 序列匹配度 | 50-70% |
| 文件大小差异 | 允许差异 | 10% |

### 硬盘保护

```bash
# 每处理100GB休息10秒
python 视频管家.py scan /path --protect 100 --rest 10
```

### 定时任务

```bash
# 每周日凌晨2点自动扫描
python 视频管家.py schedule --scan weekly --time "0 2 * * 0"

# 每天检查重复
python 视频管家.py schedule --dedup daily --time "0 3 * * *"
```

## 🐛 常见问题

### Q: 扫描很慢怎么办？
**A:** 
- 使用"简单模式"跳过AI识别
- 开启硬盘保护，避免硬盘过载
- 使用"增量扫描"只处理新文件

### Q: 人脸识别不准确？
**A:**
- 确保学习样本足够（每人至少5个视频）
- 调整识别阈值（严格模式减少误报）
- 补充学习更多样本

### Q: 去重误报怎么办？
**A:**
- 调整去重阈值（提高阈值减少误报）
- 使用"对比查看"功能人工确认
- 标记"忽略"避免再次提示

### Q: 如何迁移到新电脑？
**A:**
1. 备份数据：`python 视频管家.py 备份`
2. 复制备份文件到新电脑
3. 安装视频管家
4. 恢复数据：`python 视频管家.py 恢复`
5. 重新索引视频路径

### Q: 支持什么视频格式？
**A:** 
MP4, AVI, MKV, MOV, WMV, FLV, WebM, M4V, MPG, MPEG, 3GP, TS, M2TS, VOB

### Q: 会泄露隐私吗？
**A:** 
- 所有数据本地存储
- 人脸特征本地加密
- AI分析只发送必要信息到Kimi API
- 不上传视频文件
