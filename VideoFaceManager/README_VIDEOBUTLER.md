# VideoFaceManager 影库管家参考版

参考 [影库管家](http://www.fire-tu.com/) 设计，整合核心功能：
- ✅ **以图搜视频** - 上传照片搜索包含该人物的视频
- ✅ **语义搜索** - 用自然语言描述找视频
- ✅ **高级筛选** - 多维度过滤视频
- ✅ **AI人脸识别** - 自动识别演员
- ✅ **元数据管理** - 视频信息统一存储

## 🖼️ 以图搜视频（核心功能）

### 功能说明
上传一张人物照片，系统自动搜索视频库中包含该人物的所有视频。

### 使用方式

**命令行**
```bash
python search_engine.py "/照片/张三正面.jpg"
```

**Web界面**
```bash
python manager.py
# 浏览器打开 http://localhost:5000/search
# 上传照片 → 显示匹配结果
```

### 输出示例
```

============================================================
✓ 提取到 1 个人脸特征
📚 视频库共 150 个视频
开始匹配...

  [1/150] party_2024.mp4... ✓ 匹配 (89.2%)
  [2/150] meeting.mp4... ✗
  [3/150] wedding_video.mp4... ✓ 匹配 (85.7%)
  ...

✅ 搜索完成，找到 12 个匹配视频

============================================================
搜索结果
============================================================

1. party_2024.mp4
   匹配度: 89.2%
   匹配次数: 15次
   时间点: 23.5s, 45.2s, 78.0s, 102.3s, 156.8s

2. wedding_video.mp4
```

### 工作原理
1. 提取查询照片的人脸特征向量
2. 遍历视频库，逐帧采样
3. 计算人脸相似度（余弦相似度）
4. 返回匹配度高的视频+时间戳+截图

## 🔍 语义搜索

### 功能说明
用自然语言描述搜索视频，无需精确关键词。

### 支持语法

| 查询示例 | 说明 |
|----------|------|
| `找张三的视频` | 搜索包含张三的所有视频 |
| `有张三和李四一起的` | 两人同时出现的视频 |
| `时长超过10分钟的` | 按时长筛选 |
| `找王五，时长5分钟以上` | 组合条件 |
| `生日聚会的视频` | 关键词匹配 |

### 使用方式
```bash
python search_engine.py "找张三和李四一起出现的视频"
```

## ⚡ 高级筛选

### 筛选维度

| 维度 | 选项 | 示例 |
|------|------|------|
| 时长 | 超短/短/中等/长/超长 | <1分钟 / 1-5分 / 5-15分 / 15-30分 / >30分 |
| 人物数量 | 单人/双人/多人 | 只有1人 / 2人 / 3人以上 |
| 指定人物 | 任意人物名 | 张三、李四等 |
| 年份 | 视频年份 | 2024、2023等 |

### 使用方式（代码）
```python
from search_engine import VideoFilter

filter = VideoFilter()

# 筛选15-30分钟的长视频，包含张三
results = filter.filter(
    min_duration=900,    # 15分钟
    max_duration=1800,   # 30分钟
    persons=['张三']
)

# 获取视频库统计
stats = filter.get_stats()
print(f"总视频: {stats['total']}")
print(f"总时长: {stats['total_duration']/3600:.1f}小时")
print(f"人物数: {stats['unique_persons']}")
```


```
│  阶段1: 构建人物库                                            │
│  ├── 方式A: 反向学习 (从已分类视频)                            │
│  │      python cli_hybrid.py learn /已分类视频                │
│  │                                                           │
│  └── 方式B: 主动录入 (上传参考照片)                            │
│         python cli_hybrid.py add_photos /人物照片             │
│                                                           │
│  阶段2: 扫描视频                                              │
│  ├── python cli_hybrid.py identify /未分类视频 -o /输出       │
│  └── 自动生成标签和分类                                       │
│                                                           │
│  阶段3: 智能搜索                                              │
│  ├── 以图搜视频: python search_engine.py /照片.jpg            │
│  ├── 语义搜索: python search_engine.py "找张三的视频"         │
│  └── Web面板: python manager.py                             │
│                                                           │
│  阶段4: 纠正优化                                              │
│  └── python cli_hybrid.py correct /视频 人名 --positive       │
└─────────────────────────────────────────────────────────────┘
```

## ⚙️ 配置

编辑 `core/config.py`:

```python
# 人脸识别阈值
FACE_MATCH_THRESHOLD = 0.6      # 匹配阈值（0-1）

# 搜索采样间隔
SEARCH_SAMPLE_INTERVAL = 3      # 秒，越小越准但越慢

# 搜索结果
MAX_SEARCH_RESULTS = 20         # 最大返回结果数
SAVE_SEARCH_SCREENSHOTS = True  # 是否保存匹配截图
```
