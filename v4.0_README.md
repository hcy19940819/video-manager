## 使用Kimi API（可选）

```bash
# 方式1：环境变量
export KIMI_API_KEY="your-api-key"
python 视频管家.py web

# 方式2：命令行参数
python 视频管家.py web --kimi-key "your-api-key"
```

**效果对比：**

| 输入 | 本地解析 | Kimi解析 |
|------|----------|----------|
| "帮我找找上周拍的视频" | search + 日期参数 | 直接理解时间意图 |
| "小宝的视频都在哪" | search + 人物参数 | 更自然的回复 |
| "清理一下重复的" | dedup | 理解"清理"=去重 |

## API 列表

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/chat` | POST | AI对话（支持Kimi增强）⭐ |
| `/api/scan` | POST | 提交扫描任务（异步）|
| `/api/tasks` | GET | 获取所有任务状态 |
| `/api/videos` | GET | 视频列表 |
| `/api/thumbnail/...` | GET | 缩略图 |
| `/api/video/...` | GET | 视频流 |
